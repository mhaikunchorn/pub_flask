Geospatial Map Source: Pandas and Folium on PyCharm. Why: The initial aim was to produce a geomap that we would then use a heat map to present the data. However, the tutorials online presented python and it’s modules and libraries using Jupyter Notebook. PyCharm doesn’t have the same functionality. We were able to use the Folium module to present a map and alter the csv into a list of libraries that we then pulled out the longitude and latitude data. This allowed us to produce markers – which as you can see unfortunately are a lot. The UK are well known for the pub lifestyle. Positives: Negatives: Alternatives: If we were to repeat this exercise, I would explore the use of python on Jupyter Notebook. This would allow me to explore the option of geomapping with a heat map and present the data effectively.
